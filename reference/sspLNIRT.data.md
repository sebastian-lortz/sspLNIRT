# Precomputed sample size planning results

A nested list of precomputed
[`optim_sample()`](https://sebastian-lortz.github.io/sspLNIRT/reference/optim_sample.md)
results across a factorial grid of design conditions.

## Usage

``` r
sspLNIRT.data
```

## Format

A nested list where each element is an `sspLNIRT.object`.
